# Документация проекта

## Общее описание решения

Данный проект представляет собой математическую библиотеку на Python, предназначенную для вычисления основных геометрических характеристик фигур. Библиотека предоставляет простой и удобный интерфейс для расчета площадей и периметров, что делает её полезным инструментом для образовательных целей и простых математических расчетов.

## Описание функций

### Модуль circle.py

**Функция ```area(r)```**

Вычисляет площадь окружности по заданному радиусу.

Параметры:

- **r** - радиус окружности **(float)**

Возвращаемое значение:

- **float** - площадь окружности, вычисленная по формуле π * r²

Пример вызова:
```python

from circle imp aort area

result = area(3)
print(f"Площадь окружности: {result:.2f}")  # Выведет: Площадь окружности: 28.27

```

**Функция ```perimeter(r)```**

Вычисляет длину окружности (периметр) по заданному радиусу.

Параметры:

-   **r**  - радиус окружности **(float)**

Возвращаемое значение:

-   **float** - длина окружности, вычисленная по формуле 2 * π * r

Пример вызова:
```python

from circle import perimeter

result = perimeter(4)
print(f"Длина окружности: {result:.2f}")  # Выведет: Длина окружности: 25.13

```

### Модуль square.py

**Функция ```area(a)```**

Вычисляет площадь квадрата по заданной длине стороны.

Параметры:

-   **a** - длина стороны квадрата **(float)**

Возвращаемое значение:

-   **float** - площадь квадрата, вычисленная по формуле a²

Пример вызова:
```python

from square import area

result = area(6)
print(f"Площадь квадрата: {result}")  # Выведет: Площадь квадрата: 36

```

**Функция ```perimeter(a)```**

Вычисляет периметр квадрата по заданной длине стороны.

Параметры:

- **a** - длина стороны квадрата **(float)**

Возвращаемое значение:

- **float** - периметр квадрата, вычисленный по формуле 4 * a

Пример вызова:
```python

from square import perimeter

result = perimeter(8)
print(f"Периметр квадрата: {result}")  # Выведет: Периметр квадрата: 32

```

### Модуль rectangle.py

**Функция ```area(a, b)```**

Вычисляет площадь прямоугольника по заданным сторонам.

Параметры:

-   **a** - длина первой стороны квадрата **(float)**

-   **b** - длина второй(смежной) стороны квадрата **(float)**

Возвращаемое значение:

-   **float** - площадь прямоугольника, вычисленная по формуле a * b

Пример вызова:
```python

from rectangle import area

result = area(5, 3)
print(f"Площадь прямоугольника: {result}")  # Выведет: Площадь прямоугольника: 15

```

**Функция ```perimeter(a, b)```**

Вычисляет периметр прямоугольника по заданным сторонам.

Параметры:

- **a** - длина первой стороны квадрата **(float)**

- **b** - длина второй(смежной) стороны квадрата **(float)**

Возвращаемое значение:

- **float** - периметр прямоугольника, вычисленный по формуле 2 * (a + b)

Пример вызова:
```python

from rectangle import perimeter

result = perimeter(4, 6)
print(f"Периметр прямоугольника: {result}")  # Выведет: Периметр прямоугольника: 20

```

### Модуль triangle.py

**Функция ```area(a, h)```**

Вычисляет площадь треугольника по основанию и высоте.

Параметры:

-   **a** - длина основания треугольника **(float)**

-   **h** - высота треугольника, проведенная к основанию **(float)**

Возвращаемое значение:

-   **float** - площадь треугольника, вычисленная по формуле (a * h) / 2

Пример вызова:
```python

from triangle import area

result = area(8, 5)
print(f"Площадь треугольника: {result}")  # Выведет: Площадь треугольника: 20.0

```

**Функция ```perimeter(a, b, c)```**

Вычисляет периметр треугольника по длинам трех сторон.

Параметры:

- **a** - длина первой стороны треугольника **(float)**

- **b** - длина второй стороны треугольника **(float)**

- **с** - длина третьей стороны треугольника **(float)**

Возвращаемое значение:

- **float** - периметр треугольника, вычисленный по формуле a + b + c

Пример вызова:
```python

from triangle import perimeter

result = perimeter(3, 4, 5)
print(f"Периметр треугольника: {result}")  # Выведет: Периметр треугольника: 12

```
